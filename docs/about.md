# 关于
和网站以及我的事业相关的东西

## 做个人网站的初衷
其实原因就写在首页了，真的是just for fun。如果我觉得某件事很好玩/很酷，那么无论利弊如何都会去做的，也没什么特别的原因。

仔细思考非要说出个所以然的话，就是总有很多想法，又找不到合适的方式表达出来。最开始是写作，后来读了理工科发现自己对文字的掌控力逐渐变弱了，想表达的东西形式也更丰富，不是纯文字能做到的。

然后突然有一天，“那就做个我的网站吧！”这个想法蹦了出来，随后立即被我提上日程。实现的过程也很快，只用了一个晚上就完成了主体框架，感谢Gemini老师提供技术指导（笑），不过搭建的整个过程都是我自己完成的。

但大量的工作还在后面，填充内容才是正经事。写下这段话的时候，网站除了生活指南的几个标题几乎没有内容，也就是说这是我的第一次更新！

::: details 背景小故事
现在是2026年4月17日凌晨，我本来应该睡觉，但今天晚上要和导师开线上会议，前两天突然说要和我单独聊聊，一天都焦虑的要死担心他不收我了，不然也不会在写这个。
:::

所以大家也看出来了，写东西是我缓解压力的一个方式。有时候我会和朋友说，但基本上全是大段的思绪也不方便发消息，只能发一些碎片，所以还是要有个地方把它们归置起来。

我更愿意把这个网站视为赛博世界的家。比起现实世界的房价，一年十美元的域名费还是香很多XD

实话说，做网站本身和你正在浏览网站的行为，都是世俗意义上的“无用之事”。然而这正是我所希望的，少给人生赋予意义和价值判断，把人间当游乐场就好啦。这种观点似乎和我在其他内容中表现出的执着相悖，但其实是不矛盾的。

希望大家都能找到无用之大用。

## 语言风格
中文为主，有些表达会用英文，这一部分就是解释为什么有些看起来没必要的词句要用英文。

也很简单，就是习惯。对大多数我想表达的东西，用中文能流畅地处理。如果某个词句我更习惯用英文表达出来，或者英文能更好地传达出我的意思，那么就会使用英文。

至于为什么会有习惯用英文的字词，先问英文PPT英文作业英文考试的专业课老师们吧，已力竭XP

开玩笑的。很多学术词语就是翻译不出来它原本那个意思，只是对我来说这种效应漫延到生活用语上了，因而做出一个免责声明，这与文化立场和价值倾向没有任何关系。
